# UI Components

This file documents the reusable UI components created for the TrimTime frontend.

- `GlassCard` — frosted glass card used for cards and summary widgets. File: `frontend/src/components/ui/GlassCard.jsx`
- `Button` — primary/secondary/ghost/danger variants, supports `loading` prop. File: `frontend/src/components/Button.jsx`
- `Input` — styled input with focus ring. File: `frontend/src/components/Input.jsx`
- `Badge` — small meta pill. File: `frontend/src/components/Badge.jsx`
- `Layout` — page wrapper with `max-w-6xl mx-auto px-4 py-6 space-y-6`. File: `frontend/src/components/Layout.jsx`
- `ShopCard` — shop card used on Home page. File: `frontend/src/components/shop/ShopCard.jsx`
- `BookingSummary` — used on confirm booking page to show summary. File: `frontend/src/components/booking/BookingSummary.jsx`
- `Spinner` — small loading spinner. File: `frontend/src/components/ui/Spinner.jsx`
- `Sparkline` — tiny sparkline for compact revenue graphs. File: `frontend/src/components/ui/Sparkline.jsx`

Usage examples are in the pages under `frontend/src/pages`.
